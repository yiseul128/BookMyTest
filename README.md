# BookMyTest

## App Description
The application will allow users to:
- Book tests
- Reschedule tests
- View test results
- Sign up
- Log in, out
- Update user info

The application will allow admin users to:
- Manage (create, read, update, delete) test centres
- Manage certifications

## Languages and Technologies
- Java
- Spring Framework
- JavaScript
- React
- MongoDB
- JWT
